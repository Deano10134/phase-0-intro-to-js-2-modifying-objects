Modifying Objects

Introduction

In this lesson, we will learn how to modify objects in JavaScript.

Object Modification

We can add, update, and delete properties from objects.


// Creating an object
const person = {
    name: 'Alice',
    age: 25
};
// Adding a new property
person.city = 'New York';
// Updating an existing property
person.age = 26;
// Deleting a property
delete person.city;     
console.log(person); // { name: 'Alice', age: 26 }

Practice

Try modifying the object below:


// Create an object called 'car' with properties 'make', 'model', and 'year'
const car = {
    make: 'Toyota',
    model: 'Corolla',
    year: 2020
};
// Add a property 'color' with the value 'red'
car.color = 'red';
// Update the 'year' property to 2021
car.year = 2021;
// Delete the 'model' property
delete car.model;
console.log(car); // { make: 'Toyota', year: 2021, color: 'red' }
